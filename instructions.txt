CVM++ Compiler and VM Run Instructions
======================================

Requirements
------------

1. Windows PowerShell
2. g++ with C++17 support

The repository also includes a prebuilt cvm.exe. If cvm.exe is present, you can run the sample programs directly.


Open Project Folder
-------------------

In PowerShell:

cd C:\Users\ayush\Desktop\Code\CVM_plus_plus


Build From Source
-----------------

Use this command:

g++ -std=c++17 -I include src\main.cpp src\token.cpp src\scanner.cpp src\parser.cpp src\compiler.cpp src\vm.cpp -o cvm.exe


Run REPL Mode
-------------

.\cvm.exe

Example input:

let x = 10;
print x;
x = x + 5;
print x;
exit


Run File Mode
-------------

.\cvm.exe tests\simple_math.cvm
.\cvm.exe tests\variables.cvm
.\cvm.exe tests\booleans.cvm
.\cvm.exe tests\if_else.cvm
.\cvm.exe tests\while_loop.cvm
.\cvm.exe tests\calculator_demo.cvm
.\cvm.exe tests\truth_machine.cvm


Run Input Demo
--------------

.\cvm.exe tests\input_print.cvm

Then type any value and press Enter.


What The Program Shows
----------------------

For every script, the program prints:

1. The generated AST
2. The compiled bytecode instructions
3. The VM output

This demonstrates source code being compiled to bytecode and then executed by the stack-based virtual machine.
